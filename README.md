Fullstack blogging app, built with:
    React,
    Tanstack Start,
    Better-auth,
    Drizzle-Orm,
    Shadcn-ui and many other cool libraries.
    
You can run this project locally or check it [live](https://fullstack-blogging-app.netlify.app/)

## Installation
    Define env variables: 
    DATABASE_URL
    BETTER_AUTH_URL
    BETTER_AUTH_SECRET
    UPLOADTHING_TOKEN
    GOOGLE_APP_PASSWORD

    AND SET YOUR EMAIL in src/lib/nodemailer.ts
```bash
git clone https://github.com/cxnexion/fullstack-blogging-app
cd fullstack-blogging-app
npm install
npm run dev
```
